## ğŸš€ Steps to run the project on your machine

### Prerequisites:

Make sure you have **Node.js** and **npm** installed on your system. You can check if you already have them installed by running the following commands in the terminal:


```bash
node -v
npm -v
```

## 1. Clone the repository

```js
git clone https://github.com/AndreaAlarconValdes/shopping-app.git
cd shopping-app
```

## 2. Install dependencies

### Frontend:

```js
cd client
npm install
```

### Backend:

```js
cd server
npm install
```

## 3. Run the project

### Start Backend (API)

```js
npm run dev
```

### Start Frontend (React App)

```js
npm start
```
